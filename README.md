## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/amankumar392005/CPP_DSA.git
   ```
2. Navigate to the folder:

   ```bash
   cd CPP_DSA
   ```
3. Compile any file:

   ```bash
   g++ Day1.cpp -o output
   ./output
   ```
